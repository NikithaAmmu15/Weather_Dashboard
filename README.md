A responsive Weather Dashboard built using HTML, CSS, and JavaScript that provides real-time weather information for any city using the Open-Meteo API.

Features

Search weather by city name Displays current temperature, feels-like, humidity, wind speed & pressure Hourly forecast (next hours) 7-day weather forecast Weather icons & descriptions based on weather codes Fast, API-based real-time updates Clean and responsive UI

Technologies Used

HTML CSS JavaScript (ES6)

Open-Meteo Weather API How It Works

User enters a city name App fetches latitude & longitude using Open-Meteo Geocoding API Weather data is retrieved and displayed dynamically UI updates with icons, temperatures, and forecasts
